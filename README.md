# cloneit

`cloneit` is a `skills.sh`-style skill for cloning and adapting websites or web apps with high visual fidelity.

## Skill location

```text
skills/
  cloneit/
    SKILL.md
```

## What it does

- Actively inspects live sites with Computer Use
- Clicks through navigation, tabs, menus, and representative flows
- Captures structure, styles, interactions, and assets
- Emphasizes spacing, padding, typography, CSS fidelity, and above-the-fold composition
- Adapts an existing design language for a different product when requested

## Install

After pushing this repo to GitHub:

```bash
npx skills add https://github.com/<owner>/<repo> --skill cloneit
```

## Notes

- The skill is intentionally distributed as a single-file skill at `skills/cloneit/SKILL.md`.
- Scratch artifacts in this repo are local working files and are not required for publishing the skill.
